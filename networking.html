Networking Concepts

DNS Resolution

DNS (Domain Name System) translates our domain username.github.io into an IP address. Run nslookup username.github.io to see this in action.

IP Addressing

Our site resolves to both an IPv4 (e.g., 185.199.108.153) and IPv6 address. GitHub Pages uses Fastly's CDN, so the IP may vary by location.

HTTP vs HTTPS

Our site enforces HTTPS. HTTP traffic is automatically redirected. HTTPS uses TLS encryption to protect data in transit.

Security Headers

GitHub Pages automatically serves security headers including Strict-Transport-Security which enforces HTTPS.
